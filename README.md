# Big Integer Calculator in C++

## Overview
A C++ implementation of a Big Integer Calculator capable of performing arithmetic operations on large integers.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Exponentiation (^)

## Technologies Used
- C++
- Object-Oriented Programming
- Operator Overloading
- STL

## File Structure
- BigInteger.h
- BigInteger.cpp
- main.cpp
